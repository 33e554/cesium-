简介：
本项目是33e554个人制作的基于Cesium的项目。
Cesium 是一个开源的 JavaScript 库，专注于通过 WebGL 实现3D地理空间数据可视化。

功能：
通过导入.geojson文件实现对建筑定位，并可以通过选择拉伸字段对矢量数据进行拉伸生成白模数据，有颜色切换功能。

打开服务方法：
windows：
1.配置环境
安装Node.js和npm
安装必要的包（cesium和express）：在终端中输入npm install cesium express
2.打开服务
将该项目复制到电脑中
在该项目的根目录运行终端，输入node sever.js后可以在本地打开服务。

示例文件：
在该项目根目录中的map.json为一个标准的矢量数据文件，可通过打开该文件测试项目。
